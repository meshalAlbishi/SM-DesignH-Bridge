# SM-DesignH-Bridge

## What is the Task?
Design and programming of a two-motor circuit using H-Bridge 

## The Circuit Parts

Format of Parts -> name - (QTY)

1. Arduino Uno R3 (1).
2. H-bridge Motor Driver L293D, it's an alternative to L298n because it's not available at tinkercad (1).
3. DC Motor (2).
4. 9V Battery (1).
5. breadboard (1).
